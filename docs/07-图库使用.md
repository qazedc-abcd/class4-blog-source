# 07 图库使用指南

图库是本次重构新增的功能，用来展示班级回忆照片。

---

## 原理

- 图片文件存在 **Cloudflare R2**。
- 图片信息（标题、分组、R2 key、直链）存在 `content/gallery/manifest.yml`。
- 图库页面访问时，读取 `manifest.yml` 渲染出瀑布流。

---

## 上传图片

### 方法 1：后台上传（推荐）

1. 登录 `https://class4.zichenccc.cn/admin/`。
2. 左侧菜单 → **图库**。
3. 在"标题"和"分组"里填写信息。
4. 点击上传区域，选择照片；或直接把照片拖进去。
5. 上传完成后自动保存到 R2 并加入图库。

### 方法 2：手动编辑 manifest.yml

如果你是技术维护者，可以直接改 `content/gallery/manifest.yml`：

```yaml
- id: p0001-1234567890
  url: https://drive.zichenccc.cn/raw/classmemorial/202507/12345-xxx.jpg
  thumb: https://drive.zichenccc.cn/raw/classmemorial/202507/12345-xxx.jpg
  title: 春游合影
  desc: 高一下学期春游
  group: 春游
  r2key: classmemorial/202507/12345-xxx.jpg
  date: 2026-04-05 10:30
```

然后推送到 GitHub 或点后台同步 → 拉取。

---

## 分组

每个图片可以属于一个分组，例如：

- 高一
- 运动会
- 春游
- 毕业照

图库页面顶部会显示所有分组按钮，点击可筛选。

如果不上传分组，默认显示为"未分组"。

---

## 图片规格建议

为了访问速度快、不浪费流量：

- 单张图片大小控制在 **2MB 以内**。
- 优先上传 **jpg** 或 **webp**。
- 如果 R2 有图片处理功能，可以给缩略图加处理参数（例如 `?width=400`），并把处理后的地址填到 `thumb`。

---

## 从图库删除图片

1. 后台 → 图库，找到图片。
2. 点击图片右上角的 **×**。
3. 删除只会从 `manifest.yml` 移除记录，**不会删除 R2 里的原图**。

如果你也想删除 R2 原图，需要去 Cloudflare R2 控制台手动删除。

---

## 常见问题

### 图片上传失败

- 检查 `.env` 里的 `R2_ACCOUNT_ID`、`R2_ACCESS_KEY`、`R2_SECRET_KEY`、`R2_BUCKET` 是否填对。
- 看 `docker compose logs backend` 是否有 R2 报错。
- 检查 R2 token 是否有写权限（Read & Write）。

### 图库页面不显示图片

- 确认图片分组没有被全部隐藏。
- 检查 `content/gallery/manifest.yml` 是否被提交到 GitHub 并 pull 到服务器。
- 在后台 → 同步 → 拉取，强制同步一次。

### 手机上滑动不流畅

图库页面已经做了触屏优化：点击放大、左右滑动切换、向下滑动关闭。如果还有卡顿，可能是图片太大，建议压缩后再上传。
