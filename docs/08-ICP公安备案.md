# 08 ICP / 公安备案开关

备案信息是法律要求展示的内容。本框架把 ICP 备案和公安备案做成**独立开关**，方便随时开启或关闭。

---

## 当前备案信息（示例）

- ICP 备案：鄂ICP备2026016149号-1
- 公安备案：鄂公安网备42030402000182号

这些信息已预填在 `content/site.yml` 中。

---

## 开关方式

### 方法 1：后台开关（推荐）

1. 登录 `https://class4.zichenccc.cn/admin/`。
2. 左侧菜单 → **站点设置**。
3. 找到 **备案信息** 区域。
4. 勾选或取消勾选：
   - ICP 备案显示
   - 公安备案显示
5. 点击 **保存设置**。

### 方法 2：直接改 site.yml

```yaml
beian:
  icp:
    enable: true      # true 显示，false 隐藏
    text: 鄂ICP备2026016149号-1
    url: https://beian.miit.gov.cn/
  police:
    enable: true      # true 显示，false 隐藏
    text: 鄂公安网备42030402000182号
    code: "42030402000182"
    url: http://www.beian.gov.cn/portal/registerSystemInfo?recordcode=42030402000182
```

修改后重建站点即可生效。

---

## 什么时候关闭

- **ICP 备案**：如果域名不再备案，建议关闭，否则违反相关规定。
- **公安备案**：如果备案号过期或变更，可临时关闭。

> 注意：关闭只是前端不显示，不代表合规性发生变化。如果备案取消，请及时处理域名和服务器合规事宜。

---

## 更新备案号

如果以后换备案号：

1. 在后台 → 站点设置 → 修改 ICP 或公安备案号。
2. 公安备案的 **code** 字段只填数字部分（如 `42030402000182`），不带"鄂公安网备"字样。
3. 保存后自动重建。

---

## 备案图标

公安备案左侧有一个小图标，文件是 `frontend/static/img/police.svg`。

如果你需要换成官方的备案图标，可以替换这个文件，重新构建 Docker 镜像即可。
