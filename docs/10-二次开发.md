# 10 二次开发 / 复用为其他班级站

ClassMemorial 设计之初就是想做成一个可复用的框架。你可以把它改成其他班级、社团、家庭纪念站。

---

## 快速复用

1. 复制整个项目到新目录。
2. 修改 `content/site.yml`：
   - `title`
   - `subtitle`
   - `author`
   - `email`
   - `start_year`
   - `beian`（或关闭）
   - `nav`
3. 清空 `content/posts/` 和 `content/gallery/manifest.yml`。
4. 在 `content/pages/` 里放自己的用户协议、免责声明、关于页。
5. 修改 `.env` 中的 `SITE_TITLE`、`GITHUB_REPO`、`R2_BUCKET` 等。
6. 重新构建部署。

---

## 改主题色

打开 `frontend/static/css/style.css`：

```css
:root {
  --c-accent: #534AB7;      /* 主色 */
  --c-accent-soft: #EEEDFE; /* 浅主色 */
  --c-accent-text: #3C3489; /* 主色文字 */
}
```

改成你喜欢的颜色，保存后重新部署。

---

## 改模板布局

模板在 `frontend/templates/` 里：

- `base.html`：整体布局（头、导航、脚）
- `index.html`：首页文章列表
- `post.html`：文章详情
- `gallery.html`：图库
- `page.html`：单页
- `archive.html`：归档
- `404.html`：404 页

这些模板使用 [Jinja2](https://jinja.palletsprojects.com/) 语法。懂基础 HTML 即可改。

---

## 新增页面

如果你想新增一个独立页面，比如 `/timeline/`：

1. 在 `content/pages/` 下创建 `timeline.md`：

   ```markdown
   ---
   title: 时光轴
   permalink: /timeline/
   ---
   # 时光轴

   这里写内容。
   ```

2. 在 `content/site.yml` 的 `nav` 里加：

   ```yaml
   nav:
     - {name: 首页, link: /}
     - {name: 时光轴, link: /timeline/}
   ```

3. 保存后重建站点。

---

## 加新功能

后端在 `backend/app/`：

- `main.py`：API 路由
- `posts.py`：文章 CRUD
- `gallery.py`：图库 CRUD
- `site.py`：站点配置
- `r2.py`：R2 上传
- `sync.py`：Git 同步
- `builder.py`：静态站生成

前端后台在 `admin/app.js`。

建议你按现有模块风格添加：改文件 → 后端 API → 前端调用 → 触发重建。

---

## 添加其他评论系统

如果你不想用 Twikoo，想用 Valine、Waline、Giscus 等：

1. 在 `content/site.yml` 增加对应配置字段。
2. 在 `post.html` 里替换 Twikoo 的初始化代码。
3. 在 `admin` 的站点设置里增加表单字段。
4. 重建站点。
