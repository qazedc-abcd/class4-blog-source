# 05 后台使用指南

后台地址：`https://class4.zichenccc.cn/admin/`

用 `.env` 里设置的密码登录。

---

## 仪表盘

登录后默认显示仪表盘。可以看到：

- 文章总数
- GitHub 同步是否启用
- 最新提交 hash

快捷入口：写文章、上传图库、站点设置、同步状态。

---

## 写文章

1. 左侧菜单 → **文章** → 右上角 **+ 新建**。
2. 填写标题、正文。
3. 正文支持 **Markdown**：
   - `# 标题` 一级标题
   - `## 标题` 二级标题
   - `![描述](图片链接)` 插入图片
   - `[文字](链接)` 插入链接
   - `**粗体**`、`*斜体*`
4. 右侧是**实时预览**。
5. 分类、标签用逗号分隔。
6. 置顶权重：数字越大越靠前，0 表示不置顶。
7. 点击 **保存并发布**。

保存后，后端会自动：

- 写入 `content/posts/文章标题.md`
- 重建整个站点
- 推送到 GitHub

整个过程大概 1–3 秒。

---

## 插入图片到正文

点击编辑器里的 **插入 R2 图片** 按钮：

1. 选择本地图片（jpg/png/webp/gif）。
2. 自动上传到 Cloudflare R2。
3. 上传完成后，编辑器末尾会自动插入 Markdown：

   ```markdown
   ![filename.jpg](https://drive.zichenccc.cn/raw/classmemorial/202507/xxxxx.jpg)
   ```

你可以在插入后把这段 Markdown 剪切到正文任意位置。

> 注意：图片上传到 R2 后，不会删除本地原图。R2 里也不会自动删除，除非你手动去 R2 控制台删。

---

## 编辑已有文章

在文章列表里点击 **编辑**。

如果需要改标题，直接保存即可；如果想改文章 URL（slug），建议新建一篇再删除旧的，因为改 slug 会改变文章链接。

---

## 单页管理

**单页** 是指像 `/用户协议/`、`/免责声明/`、`/about/` 这样不进入文章列表的独立页面。

- 在左侧菜单 → **单页** 里编辑。
- 用户协议和免责声明的内容由前端协议弹窗读取，**修改后所有用户下次访问会重新弹窗确认**。

---

## 图库

左侧菜单 → **图库**：

1. 点击或拖拽图片到上传区域。
2. 填写标题、分组（如"运动会"、"春游"）。
3. 上传后，图片会同时进入 R2 和图库清单。

图库页面会自动按分组显示，支持点击查看大图、左右滑动切换。

---

## 站点设置

左侧菜单 → **站点设置**：

- 站点标题、副标题、作者、邮箱、起始年份
- ICP 备案开关 + 备案号
- 公安备案开关 + 备案号 + 备案号数字
- Twikoo 启用/关闭 + envId + CDN 地址
- 协议弹窗启用/关闭 + 倒计时秒数

修改后点 **保存设置（自动重建+推送）**。

---

## 同步

左侧菜单 → **同步**：

- **立即推送**：把 content 目录改动 push 到 GitHub。
- **从 GitHub 拉取**：从 GitHub pull 最新内容并重建站点。
- **仅重建站点**：手动触发一次静态站生成。

正常情况下不需要手动点，后台保存自动完成。同步失败时可以来这里手动处理。

---

## 首页横幅大合照

`content/site.yml` 里的 `hero` 段控制首页顶部大图：

```yaml
hero:
  enable: true
  image: https://drive.zichenccc.cn/raw/班级blog/合照-2026百日誓师.webp
  title: 26届4班纪念站
  subtitle: 愿此去繁花似锦，再相逢依旧如故
  mask_alpha: 0.35
  height: 70
```

在后台 **站点设置** 里也能编辑这段配置。不需要上传新图片时，直接把 R2 直链贴到 `image` 即可；修改后点保存，首页就会重新生成。

## 文章摘要（<!-- more -->）

和 Hexo 一样，你可以在正文里插入：

```markdown
这是首页预览里显示的内容。
<!-- more -->
这里开始是正文，不会出现在首页列表。
```

首页的文章卡片只会显示 `<!-- more -->` 之前的内容作为摘要。如果没有这个标记，系统会自动截取前 120 个字符。



左侧菜单最下面点击 **退出登录**。如果换电脑登录，请重新输入密码。

---

## 提示

- 后台没有"草稿"功能，保存即发布。
- 如果你需要临时下线某篇文章，可以编辑它但保持空标题，或直接删除。
- 不要在后台同时开两个标签页编辑同一篇文章，后保存的会覆盖先保存的。
